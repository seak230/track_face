# 🎯 Face Direction Tracker

얼굴 방향을 인식해 **Obsidian에서 마우스 클릭을 자동으로 수행**하는 Python 프로그램입니다.

---

## 💡 만든 계기

**Obsidian**을 사용할 때, 터미널 패널과 파일 탐색기(혹은 노트 패널)를 **마우스로 번갈아 클릭**해야 하는 게 너무 귀찮아서 만들었습니다.

> *"그냥 고개를 돌리면 알아서 클릭해 주면 안 될까?"*

웹캠으로 얼굴을 바라보는 방향을 실시간으로 감지하고, **정면을 보면 터미널 쪽**, **오른쪽을 보면 파일 탐색기 쪽**을 자동으로 클릭합니다.  
단, **Obsidian 창이 활성화(Foreground)된 경우에만** 마우스 제어가 작동하므로, 다른 작업 중에는 오동작하지 않습니다.

---

## ✨ 주요 기능

- 👁️ **실시간 얼굴 방향 감지** — MediaPipe Face Mesh로 코 끝의 상대적 위치를 기반으로 방향 판별
- 🖱️ **자동 마우스 클릭** — 방향이 바뀔 때 설정된 좌표로 마우스 이동 후 클릭
- 🪟 **특정 창 한정 작동** — 설정한 창(`Obsidian`)이 활성화된 경우에만 마우스 제어 실행
- 🔁 **클릭 후 커서 복귀** — 정면 클릭 후 마우스를 화면 중앙으로 자동 복귀
- 📟 **콘솔 로그** — 방향 변화 및 클릭 동작을 터미널에 실시간 출력
- 🪞 **거울 모드** — 영상을 좌우 반전해 자연스러운 셀카 뷰 제공

---

## 🛠️ 동작 원리

```
웹캠 입력 → 좌우 반전(거울 모드) → BGR → RGB 변환
         → MediaPipe Face Mesh로 468개 랜드마크 추출
         → 얼굴 외곽(뺨·이마·턱) 대비 코 끝의 상대 비율(ratio_x, ratio_y) 계산
         → 임계값 비교로 방향 판별
         → Obsidian 창 활성화 여부 확인
         → 조건 충족 시 설정된 좌표로 자동 클릭
```

### 방향 판별 기준

| 조건 | 판별 방향 | 동작 |
|---|---|---|
| `ratio_x > 0.65` | 우 (Right) | `RIGHT_CLICK_POS` 좌표 클릭 |
| 그 외 | 정면 (Front) | `LEFT_CLICK_POS` 좌표 클릭 후 커서 중앙 복귀 |

> 현재 버전은 **좌우 2방향** (정면 / 우) 제어에 집중되어 있습니다.

## ⚙️ 설정

`main.py` 상단의 설정 구역에서 좌표와 대상 창 이름을 수정할 수 있습니다.

```python
# --- [설정 부분] ---
LEFT_CLICK_POS  = (580, 140)    # 정면을 봤을 때 클릭할 좌표 (예: 터미널 패널)
RIGHT_CLICK_POS = (2250, 98)    # 오른쪽을 봤을 때 클릭할 좌표 (예: 파일 탐색기)

TARGET_WINDOW_TITLE = "Obsidian"  # 마우스 제어를 허용할 창 이름
```

- 좌표는 **본인 모니터 해상도에 맞게** 수정하세요.
- `TARGET_WINDOW_TITLE`에 다른 앱 이름을 넣으면 해당 앱에서도 사용 가능합니다.

## 🚀 설치 및 실행

### 요구 사항

- Python **3.12.0**
- 웹캠(카메라) 장치

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

주요 패키지:

| 패키지 | 버전 | 용도 |
|---|---|---|
| `mediapipe` | 0.10.14 | 얼굴 랜드마크 추출 |
| `opencv-python` | 5.0.0.93 | 웹캠 입력 및 영상 처리 |
| `pyautogui` | - | 마우스 자동 제어 |
| `pygetwindow` | - | 활성 창 이름 감지 |
| `numpy` | 2.5.0 | 수치 연산 |

### 2. 실행

```bash
python main.py
```

### 3. 종료

웹캠 영상 화면을 클릭한 후 **`ESC`** 키를 누르면 종료됩니다.  
마우스가 화면 모서리로 이동하면 `pyautogui`의 **Fail-Safe** 기능으로 자동 종료됩니다.

## ⚠️ 주의사항

- `pyautogui.FAILSAFE = True` 설정으로 마우스를 화면 **왼쪽 상단 모서리**로 이동하면 강제 종료됩니다.
- 클릭 좌표는 **반드시 본인 모니터 해상도에 맞게** 조정해야 합니다.
- 얼굴 인식을 위해 **충분한 조명**이 필요합니다.
